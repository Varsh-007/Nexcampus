# NexCampus 🎓

NexCampus is a smart campus management platform designed to simplify student life by providing a centralized system for announcements, assignment submissions, notifications, and club event bookings.

---

## 🚀 Features
- 📢 Access latest announcements
- 📝 Submit assignments easily
- 🔔 Real-time notifications
- 🎟 Book and manage club events
- 🔐 Secure user authentication

---

## 🛠 Tech Stack
- **Frontend:** React.js  
- **Backend:** Node.js, Express.js  
- **Database:** MongoDB  

---

## 📁 Project Structure

NexCampus/
├── frontend/
├── Backend/
├── README.md


## ⚙️ How to Run Locally

### 1. Clone the repository

git clone https://github.com/varsh-007/Nexcampus.git

cd NexCampus


### 2. Install dependencies

cd Backend
npm install

cd ../frontend
npm install


### 3. Run the project
Run backend

cd Backend
npm start

Run frontend

cd frontend
npm start
